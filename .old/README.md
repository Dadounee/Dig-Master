
# Dig Master

This game is a simple game where a player (you) can mine different ores across severals maps to get richier and stronger

## Compilation

Compile my game using the given makefile

If you're on linux, simply use the command:

```
  make all
```

If you're on windows, delete the file named MakeFile and rename the file "MakeFile (w)" Makefile before entering the same command. Make sure to have WinGnu32 installed.
In case you don't have WinGnu32 installed you can simply execute the given win.bat file which will compile and execute the program.
